## **Informations sur le créateur d'images exportées**
- Veuillez noter que vous ne pouvez pas définir de valeurs pour les champs **Application** et **Producer**, car [Aspose Ltd.](https://www.aspose.com) et [Aspose.PSD pour Java x.x.x](https://products.aspose.com/psd/java) seront affichés pour ces champs.
