## **Introduction**

Aspose.PSD pour Java fournit une impressionnante API de formats de fichiers PSD et PSB sans nécessiter Adobe® Photoshop® et des capacités d'exportation de niveau d'entrée pour le format de fichier AI sans nécessiter Adobe® Illustrator®. Aspose.PSD pour .NET permet de créer et de modifier les fichiers Photoshop ainsi que de mettre à jour les propriétés des calques, d'ajouter des filigranes, d'effectuer des opérations graphiques ou de convertir un format de fichier en un autre.
Ici vous pouvez trouver le guide du développeur pour travailler avec l'API Aspose.PSD.
