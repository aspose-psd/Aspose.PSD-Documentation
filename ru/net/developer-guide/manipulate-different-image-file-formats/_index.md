## **Обзор**
С помощью Aspose.PSD для .NET разработчики могут преобразовывать форматы файлов AI, PDF, PNG и многие другие в формат PSD. Следующие темы объясняют подход к загрузке существующего файла изображения и его преобразованию в формат PSD с использованием класса PsdOptions.
