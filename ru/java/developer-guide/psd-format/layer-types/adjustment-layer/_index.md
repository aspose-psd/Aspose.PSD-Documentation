## **Обзор слоев коррекции**
Эта статья предназначена для ознакомления разработчиков с API слоев коррекции Aspose.PSD для Java. Библиотека уже поддерживает (на момент написания v20.6) следующие слои коррекции Photoshop: яркость/контраст, уровни, кривые, экспозиция, оттенок/насыщенность, цветовой баланс, черно-белый, фотофильтр, смешивание каналов и инверсия. Почти все API интуитивно понятны и легко использовать. Однако есть несколько из них, на которые следует уделить больше внимания, но не волнуйтесь, они будут более подробно объяснены далее в этой статье.
