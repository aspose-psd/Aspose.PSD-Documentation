## **Обзор**
С помощью Aspose.PSD для Java разработчики могут преобразовывать файлы AI, PDF, PNG и многие другие форматы файлов в формат PSD. В следующих разделах объясняется подход к загрузке существующего изображения и его преобразованию в PSD с использованием класса PsdOptions.
