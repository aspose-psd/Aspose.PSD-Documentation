## **Приклади використання:**

**PSDJAVA-576. [Формат AI] Додано базову обробку багатосторінкових зображень AI**

{{< highlight java >}}

        String sourceFile = "src/main/resources/threePages.ai";
        String firstPageOutputPng = "src/main/resources/firstPageOutput.png";
        String secondPageOutputPng = "src/main/resources/secondPageOutput.png";
        String thirdPageOutputPng = "src/main/resources/thirdPageOutput.png";

        // Завантажити зображення AI.
        try (AiImage image = (AiImage) Image.load(sourceFile)) {
            // За замовчуванням, ActivePageIndex рівний 0.
            // Тому, якщо ви збережете зображення AI без зміни цієї властивості, буде рендерити і зберігати першу сторінку.
            image.save(firstPageOutputPng, new PngOptions());

            // Змінити активний індекс сторінки на другу.
            image.setActivePageIndex(1);

            // Зберегти другу сторінку зображення AI як зображення PNG.
            image.save(secondPageOutputPng, new PngOptions());

            // Змінити активний індекс сторінки на третю.
            image.setActivePageIndex(2);

            // Зберегти третю сторінку зображення AI як зображення PNG.
            image.save(thirdPageOutputPng, new PngOptions());
        }

{{< /highlight >}}
