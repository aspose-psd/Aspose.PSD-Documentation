## **Огляд налаштувальних шарів**
Ця стаття призначена знайомити розробників з API налаштувальних шарів Aspose.PSD для Java. Бібліотека вже підтримує (на момент написання v20.6) наступні налаштувальні шари Photoshop: яскравість/контраст, рівні, криві, експозиція, відтінок/насиченість, колірний баланс, чорно-білий, фотофільтр, мікшир каналів та інверсія. Майже всі API інтуїтивно зрозумілі та легкі у використанні. Однак є кілька з них, на які потрібно звернути більше уваги, але не хвилюйтеся, вони будуть ретельно пояснені далі в цій статті.
