## **Вступ**

Aspose.PSD для Java надає імпресивне API для форматів файлів PSD та PSB без необхідності Adobe® Photoshop® та початкові можливості експорту для формату файлів AI без необхідності Adobe® Illustrator®. Aspose.PSD для .NET дозволяє створювати та редагувати файли Photoshop, а також надає можливість оновлювати властивості шарів, додавати водяні знаки, виконувати графічні операції або конвертувати один формат файлу в інший.
Тут ви можете знайти руководство розробника для роботи з API Aspose.PSD.
