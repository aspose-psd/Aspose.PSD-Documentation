За допомогою Aspose.PSD для .NET розробники можуть конвертувати PSB в формат файлу PDF. Ця стаття показує, як експортувати/конвертувати файл PSB в формат PDF за допомогою Aspose.PSD. Aspose.PSD для .NET надає метод Load, який використовується класом [**Image**](https://reference.aspose.com/psd/net/aspose.psd/image) для завантаження файлів PSB, а також для збереження результатів. Клас [**PdfOptions**](https://reference.aspose.com/psd/net/aspose.psd.imageoptions/pdfoptions) надає параметри для створення PDF, такі як [PageSize](https://reference.aspose.com/psd/net/aspose.psd.imageoptions/pdfoptions/properties/pagesize), JpegQuality та інші. Ці параметри можна використовувати для отримання бажаного стандарту PDF.
