## **ภาพรวมของชั้นการปรับเสริม**
บทความนี้มีวัตถุประสงค์ในการทำให้ผู้พัฒนาเรียนรู้ API ของ adjustment layers ใน Aspose.PSD สำหรับ Java ไลบรารีอย่างไรก็ตาม ไลบรารีรองรับเรียบร้อยแล้ว (เวอร์ชัน 20.6 ในเวลาที่เขียน) ชั้นการปรับเสริมของ Photoshop ต่อไปนี้: ความสว่าง/ความคม, ระดับ, เส้นโค้ง, การปรับแต่ง Exposure, โทน/ความอิ่มพวกสี, สมดุลของสี, ขาวดำ, ตัวกรองภาพ, ช่วงผสมสี และการสลับ โดยจะมี API ส่วนใหญ่ที่เข้าใจง่ายและใช้ง่าย อย่างไรก็ตาม ก็ยังมีบางส่วนที่ต้องให้ความสำคัญมากขึ้น แต่อย่าต้องกังวล เนื้อหาเหล่านี้จะถูกอธิบายโดยละเอียดมากขึ้นในบทความนี้
