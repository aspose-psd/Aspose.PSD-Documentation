## **ส่งออกไฟล์ AI เป็น PSD**
โดยใช้ Aspose.PSD for Java, นักพัฒนาสามารถแปลงไฟล์ Adobe Illustrator เป็นรูปแบบ [PSD](https://wiki.fileformat.com/image/psd/) ได้ หัวข้อนี้อธิบายวิธีการโหลดไฟล์ Adobe Illustrator ที่มีอยู่และแปลงเป็น PSD โดยใช้คลาส **PsdOptions** 

ขั้นตอนในการแปลงไฟล์ Adobe Illustrator เป็น PSD ง่ายดังต่อไปนี้:

- สร้างอินสแตนซ์ของคลาส [AiImage](https://reference.aspose.com/java/psd/com.aspose.psd.fileformats.ai/AiImage) และโหลดภาพโดยใช้เมธอด Load ของคลาส Image
- สร้างอินสแตนซ์ของคลาส PsdOptions
- เรียกใช้เมธอด [AiImage.Save](https://reference.aspose.com/java/psd/com.aspose.psd/Image#save--) พร้อมกับที่อยู่ปลายทางและอินสแตนซ์ของ PsdOptions

โค้ดตัวอย่างด้านล่างแสดงวิธีการส่งออก Adobe Illustrator เป็น PSD.
