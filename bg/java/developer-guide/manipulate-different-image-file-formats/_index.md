## **Преглед**
С използването на Aspose.PSD за Java, разработчиците могат да конвертират AI, PDF, PNG и много други формати на файлове в PSD формат. Следващите теми обясняват метода за зареждане на съществуващ файл с изображение и конвертирането му в PSD чрез класа PsdOptions.
