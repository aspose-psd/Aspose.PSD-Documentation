## **Преглед на Слоевете за Настройки**
Този документ е предназначен да запознае разработчиците с API за слоеве за настройки на Aspose.PSD за Java. Библиотеката вече поддържа (v20.6 към момента на писане) следните слоеве за настройки на Photoshop: яркост/контраст, нива, криви, експонация, оттенък/наситеност, цветова баланс, черно и бяло, филтър за снимки, смесване на канали и инверсия. Почти всички API са интуитивни и лесни за използване. Въпреки това, има някои от тях, които изискват повече внимание, но не се притеснявайте, те ще бъдат обяснени подробно по-надолу в този документ.
