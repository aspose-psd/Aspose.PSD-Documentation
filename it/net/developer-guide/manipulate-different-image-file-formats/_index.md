## **Panoramica**
Utilizzando Aspose.PSD per .NET, gli sviluppatori possono convertire formati di file AI, PDF, PNG e molti altri nel formato PSD. Gli argomenti successivi spiegano l'approccio per caricare un file immagine esistente e convertirlo in PSD utilizzando la classe PsdOptions.
