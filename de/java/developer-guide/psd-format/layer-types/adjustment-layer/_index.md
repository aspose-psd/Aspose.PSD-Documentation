## **Übersicht über Anpassungsebenen**
Dieser Artikel soll Entwickler mit der Anpassungsebenen-API von Aspose.PSD für Java vertraut machen. Die Bibliothek unterstützt bereits (Version 20.6 zum Zeitpunkt der Erstellung) die folgenden Photoshop-Anpassungsebenen: Helligkeit/Kontrast, Stufen, Kurven, Belichtung, Farbton/Sättigung, Farbbalance, Schwarzweiß, Fotofilter, Kanalmixer und Umkehrung. Fast alle APIs sind intuitiv und einfach zu bedienen. Es gibt jedoch auch einige von ihnen, die mehr Aufmerksamkeit erfordern, aber keine Sorge, sie werden im weiteren Verlauf dieses Artikels ausführlich erläutert.
