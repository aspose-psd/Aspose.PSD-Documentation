## **Unterstützte PSD Ebenentypen**

- Basisebene

		Reguläre Ebenen, die an Qualität verlieren, wenn sie transformiert werden.
- Textebenen

		Ebenen, die Textdaten enthalten, einschließlich Stilen und Absätzen des PSD-Text-Editors.
- [Anpassungsebenen](/psd/de/java/layer-types/adjustment-layer/)

		Diese Ebenen wenden unterschiedliche grafische Anpassungen auf darunterliegende Ebenen an.
		
- Füllebenen

		Eine Unterart von Smart Layer, die ein Bild mit Farbe, Verlauf oder Muster füllt.
- Smartobjekte

		Ebenen, die ihren ursprünglichen Zustand speichern und bei Transformation keine Qualität verlieren.
