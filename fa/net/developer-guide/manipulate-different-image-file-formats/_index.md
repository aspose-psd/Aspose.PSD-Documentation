## **مرور**
با استفاده از Aspose.PSD برای .NET، توسعه‌دهندگان می‌توانند فرمت‌های AI، PDF، PNG و بسیاری دیگر را به فرمت PSD تبدیل کنند. موضوعات بعدی روش بارگذاری یک فایل تصویر موجود و تبدیل آن به PSD با استفاده از کلاس PsdOptions را توضیح می‌دهد.
