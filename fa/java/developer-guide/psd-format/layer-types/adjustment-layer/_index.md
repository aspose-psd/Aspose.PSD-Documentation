## **بررسی لایه‌های تنظیم در PSD**
این مقاله قرار است توسعه‌دهندگان را با رابط برنامه نویسی مبنای لایه‌های تنظیم Aspose.PSD برای Java آشنا کند. کتابخانه قبلا (در نگارش v20.6 زمان نگارش) پشتیبانی از لایه‌های تنظیم فتوشاپ زیر را دارد: روشنایی/کنتراست، سطوح، منحنی‌ها، تنظیم نوردهی، رنگ و اشباع، تعادل رنگ، سیاه و سفید، فیلتر عکس، مخلوط‌کن کانال و برعکس. تقریبا تمامی رابط برنامه نویسی آن‌ها قابل فهم و آسان است. با این حال، تعدادی دیگر از آن‌ها نیازمند تمرکز بیشتری هستند اما نگران نباشید که در این مقاله به تفصیل توضیح داده خواهند شد.
