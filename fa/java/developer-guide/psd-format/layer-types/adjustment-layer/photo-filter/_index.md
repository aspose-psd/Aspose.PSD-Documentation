# کار با لایه تنظیم فیلتر عکس فتوشاپ در جاوا

امروز خواهیم دید که چگونه می‌توانیم از Aspose.PSD برای جاوا برای اعمال لایه تنظیم فیلتر عکس به یک سند فتوشاپ موجود استفاده کنیم که کتابخانه‌ای برای دستکاری فرمت فایل PSD است.

**API لایه تنظیم فیلتر عکس** تعادل رنگی تصویر را با استفاده از رنگ‌ها تغییر می‌دهد. تصویر نتیجه پس از استفاده از یک فیلتر دوربین واقعی به نظر خواهد رسید. توجه داشته باشید که API لایه تنظیم فیلتر عکس کتابخانه به‌دلیل عدم وجود فیلترهای پیش‌تعریف شده، کمی با فتوشاپ متفاوت است. با این حال، در سایر موارد، مشابه است. به این معنا است که می‌توانید رنگ رنگ را تنظیم کنید و یکنواختی آن را تغییر دهید (چگونگی) و همچنین گزینه حفظ روشنایی را استفاده کنید.

## مرور API

API لایه تنظیم فیلتر عکس بسیار ساده‌ای است. کلاس اصلی [PhotoFilterLayer](https://reference.aspose.com/psd/java/com.aspose.psd.fileformats.psd.layers.adjustmentlayers/photofilterlayer) به عنوان نقطه ورود به این لایه تنظیم عمل می‌کند و تنها سه ویژگی عمومی دارد، به نوعی، رنگ، چگونگی و حفظ روشنایی است که با استفاده از آن تنظیمات انجام می‌شود.

## تصحیح تعادل رنگی

از آنجا که انقدر چیزی برای صحبت کردن نیست، بیایید **یک مثال از تنظیم تعادل رنگی** با استفاده از فیلتر عکس را فوراً مورد بررسی قرار دهیم. ما قصد داریم یک فیلتر گرم‌کننده (الف) را به صورت دستی به تصویر مجسمه گوزن (ب) اضافه کنیم تا تصویر را در تون‌های گرم (ج) که مطابقتی بیشتر با نگاه کردن به آن دارد، دریافت کنیم.

![مثال لایه تنظیم فیلتر عکس](photo-filter-adjustment-layer-figure-1.png)

اول از همه توجه داشته باشید که [روش کارخانه](https://reference.aspose.com/psd/java/com.aspose.psd.fileformats.psd/PsdImage#addPhotoFilterLayer-com.aspose.psd.Color-) از روش‌های [دیگر لایه‌های تنظیم](https://docs.aspose.com/display/psdjava/PSD+Adjustment+Layers) به‌خاطر عدم وجود روش پیش‌فرض (بدون آرگومان) با آن‌ها متفاوت است. بنابراین، برای اضافه کردن لایه تصحیح فیلتر عکس به داکیومنت فتوشاپ بارگذاری شده، فقط آرگومانی که رنگ است، نیاز است. بنابراین، به منظور ایجاد تأثیر فیلتر گرم‌کننده، فقط رنگ نارنجی را به عنوان آرگومان به روش کارخانه انتقال دهید، سپس چگونگی را با استفاده از تنظیم‌کننده‌های مربوطه تنظیم کنید:

    PhotoFilterLayer photoFilterLayer = psdImage.addPhotoFilterLayer(Color._fromArgb_(236, 138, 0));
    photoFilterLayer.setDensity(25);

ارزش افزوده‌ای دارد که خاصیت چگونگی مقدار پیش‌فرضی دارد که 100 است و همچنین ارزش پیش‌فرضی برای خاصیت حفظ روشنایی true است (این دلیل است که ما این گزینه را به طور صریح فعال نمی‌کنیم).

## نتیجه

در این مقاله، استفاده از API لایه تنظیم فیلتر عکس Aspose.PSD برای جاوا را مورد بررسی قرار دادیم. این ابزار ساده برای استفاده است و اجازه می‌دهد تا یک تز به یک تصویر با چگالی متغیر اضافه کند. این روشی سریع برای تنظیم تعادل رنگی کل تصویر است.
