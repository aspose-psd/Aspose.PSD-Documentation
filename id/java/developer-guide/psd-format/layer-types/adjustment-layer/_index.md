## **Ikhtisar Lapisan Penyesuaian**
Artikel ini bertujuan untuk memperkenalkan pengembang dengan API lapisan penyesuaian dari Aspose.PSD untuk Java. Perpustakaan tersebut sudah mendukung (v20.6 pada saat penulisan) lapisan penyesuaian Photoshop berikut: kecerahan/kontras, level, kurva, paparan, saturasi hue, keseimbangan warna, hitam & putih, filter foto, mixer saluran, dan inversi. Hampir semua API intuitif dan mudah digunakan. Namun, ada juga beberapa di antaranya yang perlu lebih difokuskan tetapi jangan khawatir karena akan dijelaskan secara detail lebih lanjut dalam artikel ini.
