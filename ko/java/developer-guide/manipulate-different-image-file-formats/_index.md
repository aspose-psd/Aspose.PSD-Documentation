## **개요**
Aspose.PSD for Java를 사용하면 개발자들이 AI, PDF, PNG 및 여러 다른 파일 형식을 PSD 형식으로 변환할 수 있습니다. 다음 주제에서는 기존 이미지 파일을 로드하고 PsdOptions 클래스를 사용하여 PSD로 변환하는 방법을 설명합니다.
