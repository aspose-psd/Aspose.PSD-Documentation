## **조정 레이어 개요**
이 문서는 Aspose.PSD for Java의 조정 레이어 API를 개발자에게 익히도록 의도되었습니다. 라이브러리는 이미 (작성 시점의 v20.6 기준으로) 다음 Photoshop 조정 레이어를 지원합니다: 밝기/대조, 레벨, 곡선, 노출, 색조/채도, 컬러 밸런스, 흑백, 사진 필터, 채널 믹서 및 반전. 대부분의 API는 직관적이고 사용하기 쉽습니다. 그러나 몇 가지에는 보다 더 집중해야하는 것들이 있지만 걱정할 필요가 없습니다. 이러한 내용들은 이 문서의 이후에 자세히 설명될 것입니다.
