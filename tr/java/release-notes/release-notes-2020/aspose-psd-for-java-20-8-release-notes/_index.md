{{% alert color="primary" %}} Bu sayfa, [Aspose.PSD for Java 20.8](https://downloads.aspose.com/psd/java/new-releases/aspose.psd-for-java-20.8/) için sürüm notlarını içerir. {{% /alert %}} 

|**Anahtar**|**Özet**|**Kategori**|
| :- | :- | :- |
|PSDJAVA-264|SoLdResource'un (Akıllı Nesne Katman Veri kaynağı) Desteklenmesi|Özellik|
|PSDJAVA-263|PlLdResource'un (Akıllı Nesne Katman için yerleştirilmiş katman kaynağı) Desteklenmesi|Özellik|
|PSDJAVA-262|Object Array ve Unit Array yapılarına destek eklenmesi: ObAr / UnFl imzaları|Özellik|
|PSDJAVA-265|Aspose ile kaydedilen dosyada metne odaklandıktan sonra alt çizgi ve üzeri çizili kayboluyor.|Hata|
|PSDJAVA-257|Değiştirilmiş CMYK Renk Modlu 16 bit kanalı olan PSD görüntüsünün kaydedilmesi düzeltildi.|Hata|
|PSDJAVA-268|Regresyon: Aspose.PSD 20.7.0, eski dosyalarda yazı tipi boyutlarını bozar.|Hata|
