## **Přehled úpravových vrstev**
Tento článek je určen k seznámení vývojářů s API úpravových vrstev aplikace Aspose.PSD pro Java. Knihovna již podporuje (v20.6 v době psaní) následující úpravové vrstvy Photoshopu: jas/kontrast, úrovně, křivky, expozice, odstíny/sytost, barevná vyváženost, černobílá, fotografický filtr, směšování kanálů a inverze. Téměř všechna API jsou intuitivní a snadno použitelná. Existují však i některé z nich, které vyžadují větší pozornost, ale nebojte se, budou podrobně vysvětleny dále v tomto článku.
