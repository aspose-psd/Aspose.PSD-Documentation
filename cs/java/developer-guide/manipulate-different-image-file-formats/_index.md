## **Přehled**
Pomocí Aspose.PSD pro Java mohou vývojáři konvertovat formáty AI, PDF, PNG a mnoho dalších do formátu PSD. V dalších tématech je vysvětleno postup pro načtení existujícího obrázkového souboru a jeho konverzi do formátu PSD pomocí třídy PsdOptions.
