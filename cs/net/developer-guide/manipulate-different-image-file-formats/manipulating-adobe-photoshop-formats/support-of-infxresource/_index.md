## **Podpora InfxResource**
InfxResource obsahuje údaje o míchání vnitřních prvků. Aspose.PSD podporuje InfxResource v souborech PSD. K tomu API poskytuje třídu [InfxResource](https://reference.aspose.com/net/psd/aspose.psd.fileformats.psd.layers.layerresources/infxresource).
