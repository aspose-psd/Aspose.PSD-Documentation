## **Přehled**
Pomocí Aspose.PSD pro .NET mohou vývojáři převést soubory AI, PDF, PNG a mnoho dalších formátů souborů do formátu PSD. Následující témata vysvětlují přístup k načtení existujícího souboru s obrázkem a jeho převodu do formátu PSD pomocí třídy PsdOptions.
