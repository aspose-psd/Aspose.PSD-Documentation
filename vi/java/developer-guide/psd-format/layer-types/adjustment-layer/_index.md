## **Tổng Quan về Các Lớp Điều Chỉnh**
Bài viết này nhằm mục đích giới thiệu với các nhà phát triển về API của các lớp điều chỉnh trong Aspose.PSD cho Java. Thư viện đã hỗ trợ (phiên bản v20.6 khi bài viết được viết) các lớp điều chỉnh Photoshop sau: độ sáng/độ tương phản, cấp độ, đường cong, sáng, màu sắc/mã sắc, cân bằng màu, đen và trắng, bộ lọc ảnh, trộn kênh và đảo ngược. Hầu hết các API đều trực quan và dễ sử dụng. Tuy nhiên, cũng có một số API cần hơn nhiều sự tập trung nhưng đừng lo lắng, chúng sẽ được giải thích chi tiết hơn trong bài viết này.
