## **Tổng quan**
Sử dụng Aspose.PSD cho Java, nhà phát triển có thể chuyển đổi các định dạng tệp AI, PDF, PNG và nhiều định dạng tệp khác sang định dạng PSD. Các chủ đề tiếp theo giải thích cách tải một tệp ảnh hiện có và chuyển đổi nó thành PSD bằng cách sử dụng lớp PsdOptions.
