## **Descripción**
Usando Aspose.PSD para .NET, los desarrolladores pueden convertir AI, PDF, PNG y muchos otros formatos de archivo a formato PSD. Los siguientes temas explican el enfoque para cargar un archivo de imagen existente y convertirlo a PSD usando la clase PsdOptions.
