## **Descripción general de las Capas de Ajuste**
Este artículo tiene la intención de familiarizar a los desarrolladores con la API de capas de ajuste de Aspose.PSD para Java. La biblioteca ya soporta (v20.6 al momento de escribir esto) las siguientes capas de ajuste de Photoshop: brillo/contraste, niveles, curvas, exposición, matiz/saturación, balance de color, blanco y negro, filtro fotográfico, mezclador de canales e inversión. Casi todas las API son intuitivas y fáciles de usar. Sin embargo, también hay algunas de ellas que necesitan más enfoque, pero no se preocupe, se explicarán con más detalle más adelante en este artículo.
