## **支持的操作系统**
Aspose.PSD适用于下面列出的操作系统的x86和x64版本。
### **Windows**
- Microsoft Windows 2000 专业版和服务器版（建议安装SP2）
- Microsoft Windows XP 专业版和家庭版
- Microsoft Windows 2003 服务器
- Microsoft Windows Vista
- Microsoft Windows 2008 服务器
- Microsoft Windows 2008 服务器 R2
- Microsoft Windows 7
- Microsoft Windows 8
- Microsoft Windows 10


## **支持的框架**
Aspose.PSD for .NET支持以下.NET框架：

- .NET Framework 2.0或更高版本
- .NET Compact Framework 3.5


## **开发环境**
Aspose.PSD for .NET可用于在任何以.NET平台为目标的开发环境中开发应用程序，但明确支持以下环境：

- Visual Studio 2005
- Visual Studio 2008
- Visual Studio 2010
- Visual Studio 2012
- Visual Studio 2013
- Visual Studio 2015
- Visual Studio 2017
- Visual Studio 2019
