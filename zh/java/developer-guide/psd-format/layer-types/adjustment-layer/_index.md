## **调整图层概述**
本文旨在让开发人员熟悉Aspose.PSD for Java的调整图层API。该库已经支持（截至撰写本文时的v20.6）以下Photoshop调整图层：亮度/对比度、色阶、曲线、曝光、色相/饱和度、色彩平衡、黑白、光晕滤镜、通道混合器和反相。几乎所有API都直观且易于使用。然而，也有一些需要更多注意的，但不用担心，它们将在本文中进一步详细解释。
