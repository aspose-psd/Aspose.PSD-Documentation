## **סקירת שכבות הכיון**
מאמר זה מיועד להכיר מפתחים ב-כתבי API של Aspose.PSD עבור Java. הספרייה כבר תומכת (גרסה 20.6 בזמן כתיבת המאמר) בשכבות הכיון הבאות של Photoshop: בהירות/ניגוד, רמות, עקומות, חשיפה, גוון/רוויה, שיווי צבעים, שחור ולבן, מסנני תמונה, מערבל ערוצים והיפוך. רוב API-ים הם אינטואיטיביים וקלים לשימוש. אף על פי כן, יש כמה מהם שדורשים תשומת ליבה נוספת אך אל דאגה, יתבארו בפרט נוסף במהלך המאמר.
