## **تحويل ملف PSB إلى PDF**
باستخدام Aspose.PSD لـ Java ، يمكن للمطورين تحويل تنسيق ملف [PSB](https://wiki.fileformat.com/image/psb/) إلى تنسيق ملف [PDF](https://wiki.fileformat.com/view/pdf/). توضح هذه المقالة كيفية تصدير/تحويل ملفات PSB إلى تنسيق PDF باستخدام Aspose.PSD. يوفر Aspose.PSD لـ Java الأسلوب Load الذي يوفره فئة الصور لتحميل ملفات PSB ويمكن استخدام نفس الأسلوب لحفظ النتائج. توفر فئة **PdfOptions** خيارات لإنشاء ملف PDF مثل حجم الصفحة وجودة Jpeg وغيرها. يمكن استخدام هذه الخيارات للحصول على معيار PDF المطلوب.
