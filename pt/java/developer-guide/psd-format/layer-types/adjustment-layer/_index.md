## **Visão Geral das Camadas de Ajuste**
Este artigo destina-se a familiarizar os desenvolvedores com a API de camadas de ajuste do Aspose.PSD para Java. A biblioteca já suporta (v20.6 no momento da redação) as seguintes camadas de ajuste do Photoshop: brilho/contraste, níveis, curvas, exposição, matiz/saturação, balanço de cores, preto e branco, filtro de foto, misturador de canais e inversão. Quase todas as APIs são intuitivas e fáceis de usar. No entanto, também há algumas que necessitam de mais atenção, mas não se preocupe, elas serão explicadas em detalhes mais adiante neste artigo.
