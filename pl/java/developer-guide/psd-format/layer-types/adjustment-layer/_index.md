## **Przegląd warstw dostosowujących**
Ten artykuł ma na celu zapoznanie programistów z API warstw dostosowujących Aspose.PSD dla Javy. Biblioteka już obsługuje (w wersji 20.6 w momencie pisania) następujące warstwy dostosowujące Photoshopa: jasność/kontrast, poziomy, krzywe, ekspozycja, odcień/nasycenie, balans kolorów, czarno-biały, filtr fotograficzny, mieszalnik kanałów i odwrócenie. Prawie wszystkie API są intuicyjne i łatwe w użyciu. Jednakże jest kilka z nich, które wymagają większej uwagi, ale nie martw się, zostaną one dokładnie wyjaśnione w dalszej części tego artykułu.
