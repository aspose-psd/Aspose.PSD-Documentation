# **Funkcje Aspose.PSD**
Aspose.PSD obsługuje wszystkie sekcje formatu pliku Adobe Photoshop PSD. Możesz przeczytać pełny [przegląd formatu pliku PSD](/psd/pl/przeglad-formatu-pliku-psd/) i jego funkcje
