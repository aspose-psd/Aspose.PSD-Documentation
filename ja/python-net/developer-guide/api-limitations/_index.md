## **エクスポートされた画像の作成者情報**
- **Application** および **Producer** フィールドに値を設定することはできません。なぜなら、[Aspose Ltd.](https://www.aspose.com) および [Aspose.PSD for Python via .NET x.x.x](https://products.aspose.com/psd/python-net) がこれらのフィールドに表示されるからです。
