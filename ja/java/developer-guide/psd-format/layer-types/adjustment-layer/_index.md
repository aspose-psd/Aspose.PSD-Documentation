## **調整レイヤーの概要**
この記事は、Aspose.PSD for Java の調整レイヤー API に開発者を知らせることを意図しています。 このライブラリはすでに（執筆時点のバージョンは v20.6）次の Photoshop 調整レイヤーをサポートしています: 明るさ/コントラスト、レベル、カーブ、露光、色調/彩度、カラーバランス、白黒、写真フィルター、チャンネルミキサー、反転です。 ほとんどの API は直感的で使いやすいです。 ただし、焦点が必要なものもいくつかありますが、ご心配なく、それらはこの記事で詳しく説明されます。
