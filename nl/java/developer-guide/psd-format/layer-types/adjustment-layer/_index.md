## **Overzicht van Aanpassingslagen**
Dit artikel is bedoeld om ontwikkelaars vertrouwd te maken met de aanpassingslagen-API van Aspose.PSD voor Java. De bibliotheek ondersteunt al (v20.6 op het moment van schrijven) de volgende Photoshop aanpassingslagen: helderheid/contrast, niveaus, curves, belichting, tint/verzadiging, kleurbalans, zwart-wit, fotofilter, kanaalmixer en omkeren.
